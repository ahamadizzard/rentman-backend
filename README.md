# Suit Rental Management System

A full-stack application for managing suit rentals.

## Project Structure

- `rentman-frontend/` - React frontend application
- `rentman-backend/` - Node.js backend server

## Setup Instructions

1. Frontend setup:
   cd rentman-frontend
   npm install
   npm run dev
   
2. Backend setup:
   cd rentman-backend
   npm install
   npm start
